# Introduction to C++ and G++

## 1. Introduction to C++

C++ is a general-purpose, high-performance programming language developed by **Bjarne Stroustrup** in 1979 at Bell Labs as an extension of the C programming language. Initially called "C with Classes," it was renamed C++ in 1983.

### Key Features of C++
- **Object-Oriented Programming (OOP)**: Supports classes, objects, inheritance, polymorphism, encapsulation, and abstraction.
- **Procedural Programming**: Retains the procedural capabilities of C.
- **Performance**: Offers low-level memory manipulation and high execution speed, making it ideal for system software, games, and performance-critical applications.
- **Standard Template Library (STL)**: Rich library providing data structures (vectors, lists, maps) and algorithms.
- **Portability**: Highly portable across different platforms and compilers.
- **Memory Management**: Manual memory management using `new` and `delete`, with support for smart pointers in modern C++.

### History and Evolution
- **C++98** (1998): First standardized version.
- **C++11** (2011): Major update introducing auto, lambdas, smart pointers, move semantics, and concurrency support.
- **C++14, C++17, C++20, C++23**: Continued improvements in productivity, safety, and features like modules, concepts, and coroutines.

C++ is widely used in:
- Operating systems (parts of Windows, Linux kernel contributions)
- Game engines (Unreal Engine, Unity)
- Web browsers (Chrome, Firefox)
- Embedded systems and high-frequency trading software

### Basic "Hello, World!" Program

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

### Why Learn C++?
- Strong foundation for understanding computer science concepts.
- Excellent for learning memory management and performance optimization.
- High demand in industries requiring speed and control.

## 2. G++ (GNU C++ Compiler)

**G++** is the GNU Project's C++ compiler. It is part of the **GCC** (GNU Compiler Collection) toolchain and is one of the most popular and widely used C++ compilers in the world.

### What is G++?
- A free, open-source compiler that supports multiple programming languages.
- Fully compliant with modern C++ standards (C++11, C++14, C++17, C++20, C++23).
- Available on Linux, macOS, and Windows (via MinGW or WSL).

### Installing G++

#### On Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install g++
```

#### On macOS
```bash
xcode-select --install   # Installs clang (often used interchangeably)
# Or use Homebrew:
brew install gcc
```

#### On Windows
- Use **MinGW-w64** or **MSYS2**.
- Or install **Windows Subsystem for Linux (WSL)** and use the Linux version.

### Basic Usage of G++

#### Compiling a Single File
```bash
g++ -o output_program main.cpp
```

- `-o output_program`: Specifies the name of the executable.
- `main.cpp`: Source file.

#### Running the Program
```bash
./output_program     # Linux/macOS
output_program.exe   # Windows
```

#### Common Compiler Flags

| Flag              | Description                                      |
|-------------------|--------------------------------------------------|
| `-std=c++20`      | Specify C++ standard (c++11, c++17, c++20, etc.) |
| `-Wall`           | Enable all warnings                              |
| `-Wextra`         | Extra warnings                                   |
| `-O2` / `-O3`     | Optimization levels                              |
| `-g`              | Generate debug information                       |
| `-I include_dir`  | Add include directory                            |
| `-L lib_dir`      | Add library directory                            |
| `-l library_name` | Link against a library                           |

#### Example with Flags
```bash
g++ -std=c++20 -Wall -Wextra -O2 main.cpp -o myprogram
```

### Compiling Multiple Files
```bash
g++ -o myprogram file1.cpp file2.cpp file3.cpp
```

Or use **Makefiles** for complex projects.

### Debugging with G++
```bash
g++ -g main.cpp -o debug_program
gdb ./debug_program
```

   - Qt Creator
