# Basic Structure of a C++ Program

## Overview

Every C++ program follows a well-defined basic structure. Understanding this structure is essential for writing clean, organized, and compilable code.

### Typical Structure of a C++ Program

```cpp
// 1. Preprocessor Directives (Header Files)
#include <iostream>
#include <vector>
#include <string>

// 2. Using Declarations / Namespace
using namespace std;        // Optional but common for beginners

// 3. Global Variables (Avoid when possible)
int globalVar = 100;

// 4. Function Declarations (Prototypes)
int addNumbers(int a, int b);

// 5. Main Function - Entry Point of the Program
int main() {
    // Local Variables
    int x = 5;
    int y = 10;
    
    // Statements and Logic
    cout << "Hello, World!" << endl;
    cout << "Sum = " << addNumbers(x, y) << endl;
    
    return 0;   // Return 0 indicates successful execution
}

// 6. Function Definitions
int addNumbers(int a, int b) {
    return a + b;
}
